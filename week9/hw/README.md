## 문제별 개요

### 01. 로렌츠 어트랙터 (Lorenz Attractor)

에드워드 로렌츠(1963)가 기상 모델에서 발견한 3D 혼돈 시스템입니다.

**운동 방정식:**
```
dx/dt = σ(y - x)      σ = 10
dy/dt = x(ρ - z) - y  ρ = 28
dz/dt = xy - βz       β = 8/3
```

**주요 분석:**
- 로렌츠 어트랙터 3D 시각화 및 고정점(Fixed Point) 위치 표시
- 나비 효과: 초기 조건 Δx₀ = 10⁻⁸ 차이가 지수적으로 발산하는 과정
- 리아푸노프 지수 추정 → 기상 예보의 예측 한계 정량화

### 02. 푸코 진자 (Foucault Pendulum)

레옹 푸코(1851)가 파리 팡테옹에서 지구 자전을 직접 증명한 실험입니다.

**회전 좌표계 운동 방정식:**
```
ẍ = -ω₀²x + 2Ω sin(λ) ẏ
ÿ = -ω₀²y - 2Ω sin(λ) ẋ
```
(Ω = 지구 자전 각속도, λ = 위도)

**주요 분석:**
- 위도 4곳 (북극·파리·부산·적도)에서의 진자 흔적 시각화
- 세차 주기 이론값 T = 24h / sin(λ)와 RK4 시뮬레이션 비교
- 부산(35.1°N)에서의 세차 속도: 약 15.2°/hr

### 03. 결합 진동자 (Coupled Oscillators)

두 질량이 스프링으로 연결된 계의 정규 모드와 에너지 전달 현상입니다.

**운동 방정식:**
```
m₁ẍ₁ = -(k + k₂)x₁ + k₂x₂
m₂ẍ₂ =  k₂x₁ - (k + k₂)x₂
```

**주요 분석:**
- In-phase 모드(ω₋)와 Out-of-phase 모드(ω₊) 직접 시뮬레이션
- 에너지 전달 맥놀이(beat): 이론 포락선과 RK4 결과 비교
- 결합 강도 k₂ 스캔 → 모드 분리(band gap) 시각화

### 04. 로켓 역학 (Rocket Dynamics)

가변 질량 뉴턴 법칙 + 중력 + 항력을 포함한 완전한 로켓 방정식 시뮬레이션입니다.

**운동 방정식:**
```
m(t) dv/dt = F_thrust - m(t)g(h) - ½ρ(h)v²C_D·A
dm/dt = -ṁ  (연소 중)
```

**주요 분석:**
- 치올코프스키 이상값 vs 중력·항력 손실 정량 비교
- 배기 속도 별(고체/케로신/수소) 성능 비교
- 치올코프스키 곡선 (Δv vs 질량비)
- 단단 vs 2단 로켓 Δv 비교 → 다단 분리(staging)의 필요성 검증

## AI 활용 과정

**사용 도구:** Claude Code (Claude Sonnet 4.6)

이 과제의 각 파일은 Claude와의 대화를 통해 다음 과정으로 작성되었습니다:

1. **문제 정의**: 강의 예제와 겹치지 않는 역학 주제 4가지 선정 (로렌츠/푸코/결합진동자/로켓)
2. **운동 방정식 유도**: 각 시스템의 물리 방정식을 Claude와 함께 검토
3. **코드 작성**: RK4 적분기를 핵심으로, 각 시스템에 맞는 파생 함수(`*_derivs`) 설계
4. **물리 검증**: 에너지 보존, 이론값과의 비교, 극한 케이스 확인
5. **시각화**: matplotlib GridSpec으로 결과를 다각도로 시각화

**LLM 활용의 핵심 장점:**
- 복잡한 수식(이중 진자 라그랑지안, 코리올리 가속도 벡터 분해)을 자연어로 설명하고 코드로 변환
- 물리적 직관과 수치 결과의 일치 여부를 즉시 피드백
- 시각화 레이아웃과 파라미터 선택에 대한 빠른 반복 수정
