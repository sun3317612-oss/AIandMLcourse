# PRD (Product Requirements Document)
# Week3 신경망 기초 탐색기 (Neural Networks Interactive Explorer)

---

## 1. 개요 (Overview)

### 1.1 제품 목적
Week3의 신경망 기초 개념 5가지(Perceptron, 활성화 함수, Forward Propagation, MLP, Universal Approximation)를 PySide6 GUI 앱으로 인터랙티브하게 탐색할 수 있는 교육용 도구를 제작한다.

### 1.2 배경
기존 week3의 실습 스크립트(01~05)는 터미널에서 실행 후 결과 이미지를 저장하는 방식이었다. 사용자가 파라미터를 변경하며 결과를 즉시 확인하기 어렵다. PySide6 기반 GUI로 전환하면 다음과 같은 교육적 효과를 얻을 수 있다:
- 파라미터 조작 → 즉시 시각화
- 학습 과정의 인터랙티브 관찰
- 개념 간 연결성 직관적 이해

### 1.3 대상 사용자
- 부산대 물리학과 학부생 (AI와인공지능 수강생)
- 신경망 기초를 처음 배우는 학습자

## 2. 기능 요구사항 (Functional Requirements)

### 2.1 탭 1 — 퍼셉트론 (Perceptron)

| ID | 요구사항 | 우선순위 |
|----|---------|---------|
| F1-1 | 게이트 종류 선택 (AND / OR / XOR / 모두 보기) | 높음 |
| F1-2 | 학습률(Learning Rate) 조정 (0.001 ~ 1.0) | 높음 |
| F1-3 | 에포크(Epochs) 조정 (10 ~ 10,000) | 높음 |
| F1-4 | "학습 시작" 버튼 클릭 시 실시간 학습 및 시각화 | 높음 |
| F1-5 | 결정 경계(Decision Boundary) 플롯 표시 | 높음 |
| F1-6 | 입력 → 예측 → 정답 결과 텍스트 출력 | 보통 |
| F1-7 | XOR 실패 시 "선형 분리 불가능" 메시지 표시 | 보통 |

### 2.2 탭 2 — 활성화 함수 (Activation Functions)

| ID | 요구사항 | 우선순위 |
|----|---------|---------|
| F2-1 | Sigmoid, Tanh, ReLU, Leaky ReLU 체크박스 선택 | 높음 |
| F2-2 | 함수 그래프와 미분(Gradient) 동시 표시 | 높음 |
| F2-3 | 표시 모드 선택: 함수+미분 / 함수만 / 미분만 | 보통 |
| F2-4 | 함수별 특성 정보 텍스트 출력 | 낮음 |

### 2.3 탭 3 — 순전파 (Forward Propagation)

| ID | 요구사항 | 우선순위 |
|----|---------|---------|
| F3-1 | 입력값 x1, x2 직접 입력 | 높음 |
| F3-2 | 은닉층 크기 조정 (2 ~ 8) | 보통 |
| F3-3 | 활성화 함수 선택 (ReLU / Sigmoid / Tanh) | 보통 |
| F3-4 | "순전파 실행" 버튼 클릭 시 계산 수행 | 높음 |
| F3-5 | 단계별 계산 과정 (z1, a1, z2, a2) 텍스트 표시 | 높음 |
| F3-6 | 레이어별 활성화 값 막대 차트 표시 | 높음 |

### 2.4 탭 4 — MLP (XOR 문제)

| ID | 요구사항 | 우선순위 |
|----|---------|---------|
| F4-1 | 은닉층 뉴런 수 조정 (2 ~ 20) | 높음 |
| F4-2 | 학습률, 에포크 조정 | 높음 |
| F4-3 | "XOR 학습" 버튼으로 학습 시작 | 높음 |
| F4-4 | Loss 감소 곡선 (log scale) 표시 | 높음 |
| F4-5 | XOR 결정 경계 컬러맵 표시 | 높음 |
| F4-6 | 은닉층 활성화 히트맵 표시 | 높음 |
| F4-7 | 최종 정확도, Loss 결과 표시 | 높음 |

### 2.5 탭 5 — 만능 근사 (Universal Approximation)

| ID | 요구사항 | 우선순위 |
|----|---------|---------|
| F5-1 | 근사 대상 함수 선택 (Sine / Step / Complex) | 높음 |
| F5-2 | 에포크, 학습률 조정 | 보통 |
| F5-3 | "근사 시작" 버튼 클릭 시 학습 수행 | 높음 |
| F5-4 | 뉴런 3 / 10 / 50개 결과를 3개 플롯으로 비교 | 높음 |
| F5-5 | 각 뉴런 수별 MSE 값 출력 | 보통 |

## 3. 비기능 요구사항 (Non-Functional Requirements)

| ID | 요구사항 |
|----|---------|
| NF-1 | 창 최소 크기: 1200 × 700px |
| NF-2 | 학습 후 결과 표시 시간: 5초 이내 (CPU) |
| NF-3 | 한글 UI 텍스트 정상 출력 |
| NF-4 | Matplotlib 한글 폰트 (Malgun Gothic) 자동 설정 |
| NF-5 | Windows 11 환경에서 정상 실행 |
| NF-6 | 프로그램 비정상 종료 없이 파라미터 변경 반복 가능 |

---

## 4. UX 요구사항

- 좌측: 파라미터 제어 패널 (GroupBox, 최대 너비 250px)
- 우측: Matplotlib 캔버스 (공간 대부분 차지)
- 각 탭은 독립적으로 동작
- 스타일: Qt Fusion 테마 적용

---

## 5. 제약사항

- 학습은 메인 스레드에서 수행 (XOR, 만능 근사 모두 1초 이내로 완료되므로 별도 스레드 불필요)
- GPU 의존성 없음 (NumPy만 사용)
- TensorFlow, PyTorch 미사용

---

## 6. 성공 기준

- 5개 탭 모두 정상 실행
- MLP가 XOR 100% 정확도 달성
- 뉴런 50개 Universal Approximation MSE < 0.01
- 한글 UI 정상 표시
