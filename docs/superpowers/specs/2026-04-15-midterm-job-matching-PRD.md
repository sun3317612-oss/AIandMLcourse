## 1. 제품 개요

### 무엇을 만드는가

한국 취업 시장의 실용적 정보를 모아, 사용자의 스펙(학점, 학교, 어학 등)을 입력하면 가장 적합한 회사를 추천해주는 웹 플랫폼.

### 왜 만드는가

기존 취업 사이트(사람인, 잡코리아 등)는 채용 공고를 보여주지만, "내 스펙으로 어느 회사가 현실적으로 가능한가"를 직접 알려주지 않는다. 합격자 실제 스펙 데이터 기반의 매칭 서비스가 부재하다.

### 누구를 위한가

- 취업을 준비하는 대학생 / 취준생
- 자신의 스펙 수준을 객관적으로 파악하고 싶은 사람
- 자소서 양식을 한 곳에서 빠르게 확인하고 싶은 사람

## 2. 핵심 기능

### 2.1 Google 로그인 (필수)

- Google OAuth를 통한 소셜 로그인
- 로그인 후 스펙 정보 저장 및 유지
- 매번 다시 입력하지 않아도 되는 UX

### 2.2 내 스펙 입력 & 점수화

사용자가 아래 항목을 입력하면 항목별 점수(0~100)로 시각화:

| 항목 | 입력 형태 | 점수화 방식 |
|------|-----------|-------------|
| 학점 | 숫자 + 기준(4.3/4.5) | 정규화 |
| 대학교 | 검색/선택 | 수동 Tier 정의 (Tier 1~5) |
| 학과 | 선택 (계열별) | 이공계/상경계/인문계 등 |
| 토익 | 숫자 | 990점 기준 정규화 |
| OPIc/TOEFL | 선택/숫자 | 등급별 정규화 |
| 자격증 | 다중 선택 | 종류별 가중치 |
| 인턴 경험 | 개월 수 | 정규화 |

항목별 점수 바(progress bar)로 실시간 시각화 제공.

### 2.3 AI 기반 회사 매칭

- 입력된 스펙 벡터를 PyTorch MLP 모델에 통과
- 각 회사별 매칭 점수(0~1) 반환
- 상위 5~10개 회사를 카드 형태로 표시
- 각 카드: 회사명, 로고, 매칭 점수, 산업군, 자소서 보기 버튼

### 2.4 자소서 양식 조회

- 회사 카드 클릭 시 해당 회사의 자소서 문항 즉시 표시
- 사람인 API + 회사 채용 페이지 크롤링으로 수집
- 문항별 글자 수 제한 표시

### 2.5 구독 플레이스홀더 (Polar.sh)

- 현재: 무료 사용 (매칭 횟수 제한 없음)
- UI에 "프리미엄 구독" 버튼 존재 (Polar.sh 연동 준비됨)
- 클릭 시 "준비 중" 안내 또는 기본 Polar.sh 플로우
- 발표 시 향후 로드맵으로 설명:
  - 채용 마감 알림 (이메일/푸시)
  - 더 상세한 합격자 스펙 데이터 접근
  - 무제한 매칭 횟수

## 3. 페이지 구성

### 3.1 랜딩 페이지 (`/`)
- 서비스 소개 헤드라인
- 주요 기능 3가지 간단 소개
- Google 로그인 CTA 버튼

### 3.2 스펙 입력 페이지 (`/profile`)
- 항목별 입력 폼
- 실시간 점수 바 업데이트
- "매칭 시작" 버튼

### 3.3 매칭 결과 페이지 (`/results`)
- 회사 카드 목록 (매칭 점수 높은 순)
- 필터: 산업군, 기업 규모
- 각 카드에서 자소서 모달 열기

### 3.4 마이페이지 (`/me`)
- 저장된 스펙 확인 및 수정
- 구독 상태 (현재: 무료)
- Polar.sh 업그레이드 버튼
